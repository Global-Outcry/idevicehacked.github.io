How did it start?

It all started because of Junesiphone mainly. He was a big inspiration to me. His most amazing tweaks to me were LockPlusPro and FrontPage, these both really opened my eyes to what amazing things I can do with a Jailbreak. Once I saw the capability of jailbreaking it really got me back into coding. But before I started coding tweaks I started with first with widgets on Anemone. Boots and Evelyn were both big inspirations when I am designing my widgets. So basically my repo has started with inspiration from others and see what amazing capabilities can come from jailbreaking. Also just wanting to improve on what Apple have had implemented into their devices.



Why I code.

Some questions that I get is why I code and sometimes I get asked how to start. The way I started coding was that I learned the basics of the language which I wanted to do and then I just went from there by teaching myself by just seeing what works and lots of googling. Now to start coding it is pretty easy all you need to do is first start by downloading a good text editor like sublime or if you want a WYSIWYG then use DreamWeaver. DreamWeaver has a live coding view so you can code then look up and see what you are making. The next step would be finding good tutorials online which have all the basics. Next just start coding and then you will easily see what does what.
